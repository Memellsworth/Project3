IAH Flight Delays

IAH Flight Delays is a web app where you can summarize and compare the Statistical flights delay data over in 2022 in a visual manner using our dynamic charts. We use the data from the Datasets on bureau of transportation statistics and created the plots based on yearly data for flight delays for 2022.
